# libmarpa-sys Crate (dginev fork)

Low-level FFI bindings to the
[libmarpa](https://jeffreykegler.github.io/Marpa-web-site/libmarpa.html)
C library. The libmarpa 8.6.2 source is bundled as a tarball and
built + statically linked at crate-build time.

The Rust binding file is auto-generated by `bindgen` from
`marpa.h`; this crate's `build.rs` handles tarball extraction,
`./configure && make`, and `bindgen` invocation.

This is the `[dependencies.libmarpa-sys]` carrier for the
[`marpa`](../marpa/) crate. See the parent workspace `README.md`
for the full picture.
